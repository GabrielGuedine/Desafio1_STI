# Desafio1_STI

Instruções de uso :
1. Abra o terminal e até o caminho do projeto usando o comando cd no terminal ( "cd <caminho do projeto>" )
2. Caso não tenha o PIP instalado:
	2.1. Execute o comando "python get-pip.py"
3. Caso não tenha a biblioteca pandas instalada:
	3.1. Execute o comando "pip install pandas"
4. Digite no terminal o comando "python .\main.py"
5. Siga as orientações que serão descritas no programa
